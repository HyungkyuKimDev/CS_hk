# MariaDB

## Install to Execute

- Install


```
brew install mariadb
```

- Check

```
mariadb -V
```

- Execute

```
mariadb // First Way

mysql.server start // Second way

```

## Main Command

```
// MariaDB Server Execute
brew services start mariadb 

// MariaDB Server Stop
brew services stop mariadb

// MariaDB Server Restart
brew services restart mariadb

// MariaDB Server Check
brew services list
```
