# Visual Studio Code

## Shortcuts for Mac

### Basic Expression Description
|Name | Description |
|:--- |:---|   
|⇧| Shift |   
|⌘| Command |   
|⌥| Option |   

### General
|이름 | 설명 |
|:--- |:---|   
|⇧⌘P| Show Command Palette |   
|⌘P| Quick Open, Go to File |   
|⇧⌘N| New window/instance |  
|⌘W| Close window/instance |   
|⌘| User Settings |   
|⌘K ⌘S| Keyboard Shortcuts |
