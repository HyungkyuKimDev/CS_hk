最終コミットと現状を比較し、差分を取られるようにする方法

diffとは
＞
　　最後のコミットと現状を比較・差分するために使うコマンドだ。


diffを通して比較・差分を確認する方法
＞
　　最後コミットは'HEAD'で参照されていた。
　　
　　＃ 「最後コミット」　〜　「ワークツーリー」の比較・差分を確認
　　git diff HEAD

　　＃　「最後コミット」　〜　「インデックス」の比較・差分を確認
　　git diff --cached HEAD

   ＃コマンドの結果
　　+・・・・ // 追加されたこと
　　-・・・・ // 削除されたこと
